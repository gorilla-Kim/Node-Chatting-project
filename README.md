# Node-Chatting-project

## ✅ 프로젝트 설명

Micro Services

* 기능별로 최소한으로 구현하는 서비스

Serverless 

* 서버를 구현하기 위한 소스 없이 함수단위로 필요한 기능을 제공하는 서비스

-----------

`npm i -g serverless`

* Node.js 의 serverless 프레임워크
  * 프레임워크이기 때문에 `-g`
  * 옵션을 통해 Global(전역)로 설치해주어야 한다.
